# zksync_l3_poc

Proof of Concept of what a zkSync Era's L3 would look like

## Running your base layer

1. Setup the project
    ```
    make setup
    ```
2. Run `era-test-node`
    ```
    make run-node
    ```
    ```
3. Initialize
    ```
    make init
    ```

## Running your L3

1. Run the node
    ```
    make run-l3
    ```
