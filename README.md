# zksync_l3_poc

Proof of Concept of what a zkSync Era's L3 would look like

## Running the project

1. Setup the project
    ```
    make setup
    ```
2. Run `era-test-node`
    ```
    make run-node
    ```
3. Compile the contracts
    ```
    make compile
    ```
4. Deploy the contracts
    ```
    make deploy
    ```
